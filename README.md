# 11ty demo
A slightly modified version of what appears here https://youtu.be/X3SrZuH00GQ?t=1386

  1. install deps with
      ```bash
      yarn
      ```
  1. build it with
      ```bash
      yarn build
      ```
  1. start a web server with
      ```bash
      yarn serve
      ```
  1. now you can go to http://localhost:8080/20/index.html or
     http://localhost:8080/666/index.html.
